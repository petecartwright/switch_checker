What is this?

I wanted to buy a Switch but really really hate Craigslist and eBay. Best Buy is the only major retailer (that I know of!) that publishes their in-store availability via an API. This tool hits that API every morning, then outputs the results here.

Is this accurate?

shrug emoji. I called a couple stores and confirmed they had stock, but I would not be stunned if it wasn't right 100% of the time. Also, this only tells you if a Switch was in-stock as of that morning, so if someone buys them all up t 10:01 AM, I have no way of knowing. Give them a call before you head over.

How many Switches do they have in stock?

No idea. Best Buy doesn't share quantities via the API - only if a given item is or isn't in stock.

Why don't you check more often?

Best Buy only updates their data once a day. Sorry.

Couldn't this have been generalized to check stock for all sorts of products?

Yes, but I wanted a Switch.

What's this built on?

The site is being served on a Linode VPS running Ubuntu 16.04. The site backend is nginx+Gunicorn+Flask, and the front-end is (almost completely) in React.
